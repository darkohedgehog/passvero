# Passvero Codex Documentation

This directory contains project-specific guidance for Codex.

## Files

- `AGENTS.md` — primary project instructions.
- `PROJECT_CONTEXT.md` — product goals, users and MVP scope.
- `ARCHITECTURE.md` — application boundaries and folder structure.
- `CODING_STANDARDS.md` — TypeScript, React and Next.js rules.
- `I18N.md` — supported locales and next-intl conventions.
- `UI_GUIDELINES.md` — visual language and responsive behavior.
- `SECURITY.md` — authorization, validation and data handling rules.
- `TESTING.md` — required validation and regression checks.
- `TASK_WORKFLOW.md` — how Codex should execute implementation tasks.
- `DECISIONS.md` — architectural decisions already made.
- `ROADMAP.md` — phased product roadmap.
- `DEPLOYMENT_CHECKLIST.md` — production readiness checklist.
- `TASKS.md` — current project task queue.

Codex should treat these files as project rules, not optional suggestions.
